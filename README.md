# Age and Gender Prediction using OpenCV

This project uses **OpenCV's Deep Neural Network (DNN) module** with pre-trained models to detect faces in real-time from a webcam feed and predict the **age group** and **gender** of each detected face.

---

## 📌 Features
- Real-time face detection using OpenCV's DNN face detector.
- Age prediction across predefined age buckets.
- Gender prediction (Male/Female).
- Displays predictions directly on the video feed.
- Works with any standard webcam.

---

Make sure you have the following installed:

- Python 3.7+
- OpenCV (`opencv-python`)
- NumPy
